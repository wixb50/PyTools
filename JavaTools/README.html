draw.io¶

开源的在线web画图工具(可自建服务)，功能强大，适配多种格式.

安装

$ docker run -d --restart always --name=draw -p 8080:8080 -p 8443:8443 fjudith/draw.io

使用

$ 访问: http://localhost:8080/
